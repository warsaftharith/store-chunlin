# Toko Sembako Website

Website statis React + Vite + TypeScript + Tailwind untuk profile toko sembako.

## Jalankan lokal

```bash
npm install
npm run dev
```

## Build deploy

```bash
npm run build
```

Upload project ke GitHub lalu deploy ke Vercel.

## Ubah data toko

Edit file:

```txt
src/data/content.ts
```

Ubah `storeInfo` untuk nama toko, alamat, nomor WhatsApp, jam buka, dan link Google Maps.

## Ubah foto

Ganti file di folder:

```txt
src/assets/
```

Lalu sesuaikan import di `Documentation.tsx` bila nama file berubah.
